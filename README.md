# 2023-02-DatabaseSystem
23학년도 2학기 노서영 교수님 데이터베이스 시스템
### Environment
- Mysql 8.0 in VMWare Fusion 13
- Linux : RockyLinux9
- Host : Mac Air M1
- Spring Boot


## Mini-World Problem
자신이 작성한 기록물을 계층적으로 관리하여 포트폴리오나 자신의 작업을 관리할 수 있도록 기획함.
